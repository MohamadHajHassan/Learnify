# Learnify: DevBern Final Assessment Project - EdTech Platform

## Overview

Learnify is an advanced ed-tech platform designed to provide an engaging and effective learning experience. It offers features such as user management, course catalog, interactive lessons, and progress tracking.

## Technologies

* **Frontend:** React
* **Backend:** .NET Core
* **Database:** MongoDB

## Key Features

* User Management: User registration, authentication, profile management, and role-based access control.
* Course Management: Course creation, management, search, and enrollment.
* Learning Experience: Interactive lessons and quizzes.
* Progress Tracking: User progress tracking and certifications.
* Admin Dashboard: User and course management.
